# Echo OS | John Ogletree

A high-performance, modern digital portfolio and application shell for John Ogletree, featuring a professional dark mode interface and a responsive "App Shell" architecture.

## 📱 Application Architecture

The project employs a robust **App Shell Architecture**, providing a seamless, native-like experience optimized for all device screen sizes from a single codebase.

### 🎨 Design System
- **Modern Dark Mode**: A professional, high-contrast palette based on slate and indigo tones, optimized for clarity and reduced eye strain.
- **Typography**: Leverages the **Inter** font stack for maximum readability and a clean, modern application feel.
- **App Shell**: A fixed structural layout featuring a permanent IDE-style activity bar, breadcrumb navigation, and a continuous notification ticker.
- **Scroll Logic**: Confines scrolling to the main content area, keeping primary navigation and system headers locked for a professional UI experience.

### ⚙️ Core Protocols
- **Fluid Scaling**: Uses modern CSS `clamp()` and relative units to ensure the interface scales perfectly from mobile to 4K displays.
- **Vanilla Performance**: Built entirely with Vanilla HTML5, CSS3, and JavaScript for zero-latency execution.
- **Dynamic Versioning**: Real-time synchronization of the system version with the browser's local clock.
- **Stealth Links**: Implements a link-cloaking protocol to hide browser status bar URLs, maintaining the virtual OS immersion.
