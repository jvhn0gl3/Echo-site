## Project Memories

- **Automated Deployment:** When changes are ready, run `./deploy.sh "<summary>"`. This script will stage all changes, commit with a descriptive message, and push the changes.
    - **Important Note:** As per user instructions:
        - GitHub releases will NOT be created automatically.
        - Git tags will NOT be created or pushed.
- Prompt deployment is recommended for changes affecting the 'production/' directory.
