# Echo-site Production (Live)

This directory contains the live, stable version of Echo-site.

## Deployment Workflow
**Important:** Do not apply changes directly to this directory. All updates must first be verified in the [Preview (Staging) Environment](../preview/README.md).

## Files
- `index.html`: The main entry point for the website.
- `style.css`: Contains the visual styling for the site.
- `master.js`: The primary JavaScript logic for the production environment.

## Usage
The `index.html` file serves as the landing page, featuring a welcome header and linking to the necessary assets for styling and interactivity.
