Development Roadmap

Building a more resilient infrastructure

🔄 IN PROGRESS

Multi-Source Image Failover

Implementing automatic fallback for image assets across multiple CDN providers and local paths.

  • Relative path support (/images/logo.png)
  • Primary CDN: Cloudflare Pages
  • Secondary CDN: GitHub Pages
  • Tertiary: Netlify CDN (planned)
  • Local storage cache fallback (planned)
🔧 IN PROGRESS

Asset Health Monitoring

Real-time health checks to detect broken images and automatically switch to working sources.

  • Failed load detection
  • Automatic source rotation
  • Latency-based source selection
  • User dashboard for asset status
📋 PLANNING

Geo-Aware CDN Routing

Serve images from the closest CDN based on user's geographic location for faster loading.

  • Region detection via Cloudflare
  • US East: Cloudflare
  • Europe: Bunny.net
  • Asia: Fastly
