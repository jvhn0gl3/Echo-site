# CSS Schema | John Ogletree

A shareable CSS database and design system repository. This schema provides professional, high-performance stylesheets designed for modern web applications and virtual OS interfaces.

## 📦 Usage

### 1. CDN (Recommended)
You can inject the styles directly into your HTML using the jsDelivr CDN. This ensures fast delivery and automatic versioning.

```html
<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/jvhn0gl3/css-schema/echo-os.css">
```

### 2. Direct Download
Download the `echo-os.css` file and include it in your project locally.

```html
<link rel="stylesheet" href="./echo-os.css">
```
